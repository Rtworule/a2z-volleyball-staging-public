# A to Z Volleyball Center Admin Settings UI

Task `t_ac1230db` implements the administrator settings interface for A to Z Volleyball Center.

## Included

- Day-by-day operating hour controls.
- Closure creation and removal for all resources, courts, or gym trainers.
- Fixed reservation creation and removal for seasonal/team schedules.
- Pricing controls for court bookings, trainer bookings, and fixed reservation deposits.
- Scheduler-compatible settings payload preview.

## Verify

```bash
npm test
```

Open `index.html` in a browser to use the UI directly.
